📝 Manual de uso

1️⃣ Clic en lienzo o botón para crear nodos.
2️⃣ Arrastra nodos para moverlos.
3️⃣ Usa "Agregar arista" para conectar nodos.
4️⃣ Puedes definir peso y tipo (uni, bi, bucle).
5️⃣ El sistema evita solapamientos automáticamente.
6️⃣ Ver matriz abre página dinámica.
7️⃣ Guardado automático con localStorage.
8️⃣ Puedes exportar/importar JSON.
9️⃣ Borrar lienzo requiere confirmación.
10️⃣ Todo es editable en cualquier momento.
